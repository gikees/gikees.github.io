Research Interests

My goal is to build AI capable of understanding, reasoning, and planning with a human-like "commonsense" of the world.

Projects

CVJutsu GitHub

Webcam app that recognizes Naruto hand seals using MediaPipe hand tracking and a RandomForest classifier. Matches sequences of the 12 zodiac seals to known jutsu and triggers animated visual effects. Includes a built-in training pipeline for custom models. Built with Python, OpenCV, scikit-learn, and PyQt6.

SafeFlame GitHub

Real-time AI kitchen safety monitor using YOLOv8, CV heuristics, and a local LLM (Llama 3.1 via Ollama) to detect hazards and provide contextual safety advice. Features voice alerts and a live dashboard. Runs fully locally.
